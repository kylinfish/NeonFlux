概述

NeonFlux 是一個 Chrome 擴充功能，致力於保護使用者隱私。本隱私政策說明我們如何收集、使用和保護您的資料。

資料收集

NeonFlux 僅收集以下資料：

  • 瀏覽記錄：您訪問的網站 URL 和標題，用於分類和顯示
  • 快速連結：您手動新增的書籤和快速連結
  • 使用者設定：您在設定中配置的偏好設定（語言、排序方式、分類規則等）
  • Favicon：網站圖標，用於視覺識別

資料使用

我們收集的資料僅用於以下目的：

  • 根據您的規則自動分類瀏覽記錄
  • 提供搜尋和篩選功能
  • 在新分頁中顯示分類的連結
  • 儲存您的使用者偏好設定
  • 提供快速連結和其他功能

資料儲存

本機儲存：所有資料儲存在您的本機裝置上，使用 Chrome 的 storage API。我們不會將任何資料上傳到我們的伺服器或第三方伺服器。

如果您啟用了 Chrome 同步功能，您的設定可能會根據 Google 的隱私政策同步到您的 Google 帳戶。

權限說明

NeonFlux 需要以下權限：

  • history：讀取您的瀏覽記錄，用於分類和顯示
  • storage：儲存您的設定、快速連結和分類規則
  • tabs：獲取當前標籤資訊，用於快速連結功能

我們不會修改、刪除或上傳您的瀏覽記錄。

第三方服務

NeonFlux 可能使用以下第三方服務來提取 Favicon：

  • DuckDuckGo Icon Service：用於獲取網站圖標
  • Google Favicon API：備用 Favicon 來源

這些服務可能會根據其隱私政策收集您的 IP 地址和請求資訊。我們建議您查閱這些服務的隱私政策。

使用者權利

您對您的資料擁有完全控制權：

  • 您可以隨時查看和匯出您的資料
  • 您可以刪除任何快速連結或設定
  • 卸載擴充功能將刪除所有本機儲存的資料
  • 您可以禁用任何權限或功能

聯絡我們

如果您對本隱私政策有任何疑問，請通過 Chrome Web Store 的評論或反饋功能與我們聯絡。
